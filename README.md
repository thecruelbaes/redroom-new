# RedRoom Studio — лендинг

Продающий лендинг школы музыки в Новороссийске. Фокус — индивидуальные уроки барабанов и
гитары (вторично: репетиционная точка, звукозапись). Стек: **Next.js 15 + Tailwind v4**.

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # прод-сборка
```

## Главный оффер

«Подпишись на Telegram-канал [@redroomstudio](https://t.me/redroomstudio) → первое занятие
бесплатно». Этот хук — в hero, в большом CTA-блоке (`#trial`) и в плавающей кнопке на мобиле.

## Telegram-бот заявок

Форма (`#contact`) шлёт заявку в `POST /api/lead`, оттуда — в Telegram (как в Bronx).
Переменные окружения (см. `.env.example`):

- `TELEGRAM_BOT_TOKEN` — токен бота от @BotFather
- `TELEGRAM_CHAT_ID` — куда слать заявки (личка/группа)
- `NEXT_PUBLIC_SITE_URL` — прод-URL для sitemap/robots/canonical
- `NEXT_PUBLIC_YANDEX_VERIFICATION` — код Яндекс.Вебмастера (необязательно)

Без токена форма работает в dev-режиме: заявка логируется в консоль и возвращает успех
(лид не теряется при разработке). Rate-limit 5 заявок/мин с IP, ввод экранируется, honeypot.

## Дизайн

Cinematic dark: OLED-чёрный `#0A0A0B` + кроваво-красный `#E10600` + фиолетовый сценический
`#A855F7`. Шрифты Oswald (заголовки) + Inter (текст), оба с кириллицей. Подробно — в
[`DESIGN.md`](DESIGN.md). Изображения-оригиналы в `images/`, рабочие — в `public/images/`.

> ⚠️ `public/images/teacher-sergei.jpg` — 6.7 МБ. `next/image` оптимизирует на отдаче, но перед
> продом исходник лучше пересжать (≤ ~400 КБ).
