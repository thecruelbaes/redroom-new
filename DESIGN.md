# RedRoom Studio — дизайн-система лендинга

> Источник: ui-ux-pro-max (design-system) + правки под кириллицу и бренд.
> Стек: Next.js 15 (App Router) + Tailwind. Язык: ru. Цель: захват лида (заявка / подписка на TG).

---

## Концепция

Cinematic dark — «ты стоишь за кулисами рок-клуба». Полноэкранные живые фото
(барабаны/гитара в красно-фиолетовом свете и дыму), крупная условно-плакатная
типографика, мягкие scroll-reveal анимации. Это **не неон-киберпанк** — благородный
кинотёмный, где свет даёт сама фотография, а UI почти чёрный и не спорит с ней.

**Паттерн (ui-ux-pro):** Hero + Social Proof + CTA + Lead Magnet form.
**Стиль:** Dark Mode (OLED) база + Social-Proof (отзывы/преподы выводят на доверие).
Scroll-storytelling — дозированно, с `prefers-reduced-motion` фолбэком.

## Типографика (кириллица обязательна)

- **Display / заголовки:** `Oswald` (600–700) — condensed, плакатный, есть кириллица.
  Замена дефолтному Bebas Neue, который кириллицу не тянет.
- **Body / интерфейс:** `Inter` (400–600) — чистый, отличная кириллица, читаем на мобиле.
- Заголовки — uppercase + letter-spacing для «афишного» эффекта.
- Body line-height 1.6, длина строки 65–75 симв.

```
@import url('https://fonts.googleapis.com/css2?family=Oswald:wght@400;500;600;700&family=Inter:wght@400;500;600;700&display=swap');
```
(в Next.js — через `next/font/google`, subsets: `['latin','cyrillic']`.)

## Анимации (вкусно, но не тошнит)

- **Scroll-reveal:** Intersection Observer → fade-up (translateY 24px → 0, opacity), stagger 60–80ms по карточкам.
- **Hero parallax:** фон двигается медленнее контента (transform translateY, `will-change`).
- **Стат-каунтеры:** count-up при появлении в вьюпорте.
- **Hover:** карточки — рост glow/бордера (НЕ scale, чтобы не дёргать лэйаут), 200ms.
- **CTA:** мягкая пульсация свечения.
- Микро-тайминги 150–300ms, transform/opacity only.
- **`prefers-reduced-motion: reduce`** → всё мгновенно появляется, parallax off.

## Технические правила (из чек-листа)

- SVG-иконки (Lucide), не эмодзи. Логотипы брендов — Simple Icons.
- `cursor-pointer` на всех кликабельных. Видимый focus-ring.
- Картинки: `next/image`, WebP, srcset, lazy, alt на русском. Сжать `Sergei_prepod_gitara.JPG` (6.7МБ!).
- Адаптив 375 / 768 / 1024 / 1440. Без горизонтального скролла.
- SEO/безопасность/качество — по методичкам `websites/tools/*` (ship-gate перед сдачей).
- Бот: `/api/lead` (rate-limit, экранирование, env `TELEGRAM_BOT_TOKEN`/`TELEGRAM_CHAT_ID`) — порт из Bronx.
